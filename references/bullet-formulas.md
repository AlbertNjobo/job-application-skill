# Bullet Point Formulas

## Technical Bullet Formula

**[Action Verb] + [Technical What] + [Scale/Impact] + [Technology Used]**

### Examples by Role

**Software Engineer:**
- Designed and deployed 5 Node.js microservices handling 50K requests/minute, reducing system coupling
- Optimized PostgreSQL queries and implemented Redis caching, reducing API latency by 60% for 100K DAU
- Led migration to Kubernetes, achieving 99.99% uptime and reducing infrastructure costs by 35%

**Data/ML Engineer:**
- Built data pipeline processing 100M+ events daily using Kafka and Spark, reducing latency from hours to minutes
- Implemented anomaly detection model catching 95% of data issues before downstream impact
- Designed data warehouse schema enabling self-service analytics for 50+ business users

**DevOps/SRE:**
- Implemented IaC using Terraform, reducing provisioning time from 2 days to 30 minutes
- Built monitoring system with Prometheus and Grafana, reducing MTTR from 4 hours to 30 minutes
- Automated deployment pipeline with GitHub Actions, enabling 50+ daily deployments with zero downtime

**IT Support/Automation:**
- Automated procurement workflows on Power Platform, reducing approval cycle times by 60% for 25+ staff
- Deployed Freshdesk helpdesk with SLA workflows, resolving 200+ incidents in first 3 months
- Administered Microsoft 365 and Active Directory for 500+ users, maintaining 99.5% uptime

## Metrics to Include

| Category | Examples |
|----------|----------|
| Scale | "serving 500K DAU", "handling 10K req/s", "processing 50TB daily" |
| Performance | "reduced latency from 500ms to 200ms", "improved by 60%" |
| Efficiency | "cut deployment time from 2 hours to 15 min", "reduced costs by 40%" |
| Business | "drove $XM revenue", "improved conversion by 25%", "increased DAU by 30%" |

## What NOT to Include in Bullets

- "Worked on..." (vague)
- "Helped with..." (passive)
- "Responsible for..." (duty, not achievement)
- Skills bars or ratings (breaks ATS)
- Every technology you've touched once
- Microsoft Office (assumed)
